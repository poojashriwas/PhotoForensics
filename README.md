# 🔒 Photo Forensics & Geolocation Identifier

A professional Digital Image Forensics web application built using **Python** and **Flask**. The application helps investigators analyze images for metadata, GPS location, tampering, and digital evidence.

---

## 📌 Features

### ✅ Completed
- Image Upload
- Image Preview
- Flask Web Application
- Responsive User Interface

### 🚧 Upcoming
- EXIF Metadata Extraction
- GPS Coordinate Detection
- Google Maps Integration
- Error Level Analysis (ELA)
- Image Hash Generation (MD5 & SHA256)
- Digital Forensics Report (PDF)
- Risk Score Analysis

---

## 🛠 Tech Stack
